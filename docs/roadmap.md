# Roadmap

## Near Term (0.1.x)

- Complete manual smoke validation in IDEA and Rider with screenshots.
- Finalize marketplace assets and listing metadata.
- Harden app-server v2 integration with additional UI affordances for model/app/skills APIs.
- Reduce remaining deprecated IntelliJ API usage reported by Plugin Verifier.

## Mid Term (0.2.x)

- Expand parity with VS Code extension workflows where they are compatible with JetBrains UX.
- Add richer troubleshooting and self-test diagnostics in settings/tool window.
- Improve onboarding defaults and contextual help inside the plugin.

## Long Term (1.x)

- Mature full feature parity subset and stability guarantees for supported IDEs.
- Formalize maintenance cadence and support SLA expectations.
- Strengthen CI/reporting for release confidence and regression tracking.
