# Support

## Getting Help

- GitHub issues: open a bug report or feature request in this repository.
- Diagnostics: use `Codex (Unofficial): Report Issue` in the IDE for a support snapshot.

## Before Filing an Issue

1. Confirm your IDE is 2025.2+ and `JAVA_HOME` is JDK 21 for local builds.
2. Verify Codex CLI path/settings in `Settings > Tools > Codex`.
3. Capture relevant logs:
   - IDE log: sandbox `idea.log`
   - Diagnostics panel output
   - Build/test command output

## Security Issues

For vulnerability disclosure, follow [SECURITY.md](SECURITY.md) and do not file public
issues.
